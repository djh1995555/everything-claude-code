# 质量门命令

按需为文件或项目范围运行 ECC 质量管道。

## 用法

`/quality-gate [path|.] [--fix] [--strict]`

- 默认目标：当前目录（`.`）
- `--fix`：允许在配置的地方进行自动格式化/修复
- `--strict`：在支持的地方因警告而失败

## 管道

1. 检测目标的工具/语言。
2. 运行格式化程序检查。
3. 在可用时运行 lint/类型检查。
4. 生成简洁的补救列表。

## 注释

此命令镜像钩子行为，但由操作员调用。

## 参数

$ARGUMENTS:
- `[path|.]` 可选目标路径
- `--fix` 可选
- `--strict` 可选
