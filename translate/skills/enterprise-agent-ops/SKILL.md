# 企业级代理运维

将此技能用于云托管或持续运行的代理系统，这些系统需要超越单个CLI会话的操作控制。

## 操作领域

1. 运行时生命周期（启动、暂停、停止、重启）
2. 可观测性（日志、指标、追踪）
3. 安全控制（范围、权限、终止开关）
4. 变更管理（部署、回滚、审计）

## 基线控制

- 不可变部署工件
- 最小权限凭据
- 环境级机密注入
- 硬超时和重试预算
- 高风险操作的审计日志

## 要跟踪的指标

- 成功率
- 每个任务的平均重试次数
- 恢复时间
- 每个成功任务的成本
- 故障类别分布

## 事件模式

当故障激增时：
1. 冻结新部署
2. 捕获代表性追踪
3. 隔离失败路径
4. 用最小的安全变更进行补丁
5. 运行回归+安全检查
6. 逐步恢复

## 部署集成

此技能与以下技术配合使用：
- PM2工作流
- systemd服务
- 容器编排器
- CI/CD门控
