# 投资者外联

撰写简短、个性化且易于采取行动的投资者沟通内容。

## 何时激活

- 撰写给投资者的冷邮件
- 起草暖介绍请求
- 在会议后或无响应时发送跟进邮件
- 在筹款过程中撰写投资者更新
- 根据基金投资主题或合作伙伴适配性调整外联内容

## 核心规则

1. 为每封外发消息添加个性化内容。
2. 保持请求低摩擦。
3. 使用证据，而非形容词。
4. 保持简洁。
5. 绝不发送可发给任何投资者的通用文案。

## 冷邮件结构

1. 主题行：简短且具体
2. 开场白：为什么特别联系这位投资者
3. 推介：公司做什么，为什么是现在，哪些证据重要
4. 请求：一个具体的下一步
5. 签名：姓名、职位，如有需要可添加一个可信度锚点

## 个性化来源

引用以下一个或多个来源：
- 相关的投资组合公司
- 公开的投资主题、演讲、帖子或文章
- 共同联系人
- 与投资者关注点清晰的市场或产品适配性

如果缺少这些背景信息，询问用户或说明草稿是等待个性化的模板。

## 跟进节奏

默认：
- 第 0 天：初始外发
- 第 4-5 天：简短跟进，包含一个新数据点
- 第 10-12 天：最终跟进，干净收尾

除非用户想要更长的序列，否则之后不要再继续催促。

## 暖介绍请求

让介绍人的工作更轻松：
- 解释为什么这个介绍合适
- 包含可转发的文案
- 保持可转发文案在 100 词以内

## 会后更新

包括：
- 讨论的具体事项
- 承诺的答案或更新
- 如有可用，一个新的证据点
- 下一步

## 质量门

交付前：
- 消息已个性化
- 请求明确
- 没有废话或乞求语言
- 证据点具体
- 字数保持紧凑
